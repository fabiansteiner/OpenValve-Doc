# Hose system

The hose system should be designed for low resistance.

Use a relatively large main line, for example a **1/2" hose**, so water can flow with as little resistance as possible. From this main line, the water can then be distributed to individual plants using micro tubing, such as **1/4" hose** or **4 × 6 mm tubing**.

The basic idea is:

```text
large main line → smaller distribution tubes → emitters
```

This helps the limited pressure from the water container reach the emitters more effectively. A common mistake is to use **1/4" hose** or **4 × 6 mm tubing** for both the main line and the branch lines. Depending on the emitters used, this can result in uneven water distribution, with the first emitters receiving significantly more water than those farther down the line.
