# State: Off

Switching to this state turns the device off and closes the valve.

OpenValve does not perform normal soil moisture based irrigation while it is turned off.

## State transitions

Use this table to see where OpenValve can go from this state.

| Button action | Next state | What happens |
| --- | --- | --- |
| Very long press | [Display Battery Level](./display-battery-level) | Turn on OpenValve. |
